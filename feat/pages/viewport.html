# 移动端适配方案

# devicePixelRatio

能够返回当前显示设备的物理像素分辨率与 CSS 像素分辨率的比率

  • 物理像素即设计稿的像素
  • css 像素即实际的px像素

例如 iphone6的设计稿 750 * 1334，这为物理像素

实际 css 尺寸为 350 * 667
